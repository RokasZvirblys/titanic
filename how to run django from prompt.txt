1. Isijungti anaconda prompt
2. cd i faila kur yra projektas pvz cd Web Development\Django
3. Galima susikurt nauja faila pvz mkdir views_urlzzzzz(projekto pav)
* Jei reikia paeiti atgal - cd ..
4. Pirma karta reikia daryti django-admin startproject projekto pavadinimas - susikuria dar vienas failas tame projekto faile
5. Projekto faile turejo susikurt manage.py
6. python manage.py runserver
7. Ismeta URL, paprastai buna http://127.0.0.1:8000
8. Tam starproject sukurtam faile reikia sukurti pvz views.py, jame kaip pvz galima pasidaryti:

from django.shortcuts import render

def home(request):
    return render(request, 'index.html')

def about(request):
    return render(request, 'about.html')

9. Tada urls.py reikia pazymeti ta home:

from django.contrib import admin
from django.urls import path
from . import views

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', views.home),
    path('about/', views.about)
]
